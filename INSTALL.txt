📋 REQUISITOS
─────────────────────────────────────────────────────────────
✓ Node.js 18+ 
✓ npm ou yarn
✓ Conta Supabase (grátis)
✓ Git

🚀 INSTALAÇÃO RÁPIDA (5 MINUTOS)
─────────────────────────────────────────────────────────────

1️⃣ INSTALAR DEPENDÊNCIAS
   
   npm install

2️⃣ CONFIGURAR SUPABASE

   a) Criar projeto em https://supabase.com
   b) Copiar URL e chave anon em Settings → API
   c) Executar SQL em SQL Editor:
      - Abrir arquivo: supabase-schema.sql
      - Copiar todo conteúdo
      - Colar no SQL Editor
      - Clicar "Run"

3️⃣ CONFIGURAR AMBIENTE

   cp .env.local.example .env.local
   
   Editar .env.local com suas credenciais:
   
   NEXT_PUBLIC_SUPABASE_URL=https://xxx.supabase.co
   NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJxxx...
   NEXT_PUBLIC_VENDR_PIX_KEY=11999887766
   NEXT_PUBLIC_TZ=America/Sao_Paulo

4️⃣ EXECUTAR

   npm run dev
   
   Abrir: http://localhost:3000

5️⃣ PRIMEIRO ACESSO

   a) Cadastrar-se em /login
   b) Configurar empresa em /onboarding
   c) Cadastrar produtos em /produtos
   d) Cadastrar vendedores em /vendedores
   e) Atribuir kit diário

✅ PRONTO! Sistema rodando.

─────────────────────────────────────────────────────────────

📱 TESTAR OFFLINE

1. Abrir DevTools (F12) → Network → Offline
2. Registrar venda em /vendedor/venda
3. Ver "Venda salva offline"
4. Voltar Online
5. Ver sincronização automática

🐛 PROBLEMAS COMUNS

"Cannot find module 'next'"
→ Execute: npm install

"User not found"
→ Cadastre-se primeiro em /login

"RLS error"
→ Verifique se executou o SQL schema completo

"Offline não funciona"
→ PWA requer HTTPS ou localhost
